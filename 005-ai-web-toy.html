// The Brief

The brief was constraint. Most brief generators produce expansive lists. This one produces structure: a problem, a user truth, a north star, a success metric. Four outputs. Enough to think from, not enough to hide behind.

Six brief types — Error Recovery, First Use, Loading State, Empty State, AI Uncertainty, Destructive Action — because those are the moments where interface design matters most. The happy path doesn't need a brief.

The streaming output is intentional. Watching the brief form at character pace slows the designer down. The constraint arrives word by word. By the time it's done, you've already started thinking.
