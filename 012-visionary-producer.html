02 — The Loop reason, act, observe

It drafts, then asks.

A ReAct loop: the agent thinks, calls one tool, reads the result, and repeats — narrating each step into a live thinking log. Before it locks anything, it stops for the producer.

03 — Control the agent proposes; the person decides

Built to be supervised.

A creative tool still commits work that costs money. Three decisions keep the producer in command.

The Autonomy Dial
A slider between Advisory — chat only — and Active, where the agent runs tools. The producer decides how much rope it gets.
Producer-in-the-loop
Locking a package pauses the loop. Nothing reaches the production board until a person reviews and approves it.
One pattern, two domains
The agent loop is identical to Sentinel Triage's. Only the tools and the surface change — proof the architecture travels.
In review — what I'd do differently
  • The script parser is heuristic — it expects screenplay-formatted text, and degrades gracefully on freeform input.
  • Casting archetypes are reference vibes, not a real talent database. The structure is built to plug one in.
  • The next step is an export — a breakdown sheet a coordinator could actually hand to a department.
